# 🏛️ LV DOJ Paperwork System

A simple DOJ system for FiveM that allows officers to file cases and judges to look up criminal records. Includes Discord webhook integration.

---

# 📦 Features

- 📁 File DOJ cases in-game
- 🔍 Lookup criminal records
- 📡 Discord webhook logging
- 🗄️ Database storage (oxmysql)
